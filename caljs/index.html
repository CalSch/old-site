Cal.js

Cal.js is my own JavaScript library. It's just a bunch of functions that make things easier and other stuff mashed together.

Docs

Show/Hide:

Show/Hide is a thing I made where you can have hidden text like this: This is hidden by default. If you want to have this in your website, make sure the showHideEnabled variable is set to true in Cal.js, for anything you want to hide just add show-hide to the class list of the element.

collToList():

collToList() is a function that takes in an HTMLCollection object and gives out a list of all the elements inside it.

CustomElement():

CustomElement() is a function that lets you make custom elements, you give it the tag name (a string), the attributes (a list), and a function. Example: if you wanted a custom element called repeat, and there was an attribute called times, and it would reapeat the text in it that many times, the function would be this: CustomElement("repeat",["times"], function(element,args){
  element.innerHTML=element.innerHTML.repeat(args.times);
});
